# Paperclip Company Ops

Paperclip Company Ops is a SaaS workspace for Paperclip AI.

This repository is a public documentation project for Paperclip Company Ops. Its structure follows the public documentation pattern used by [MiroFish](https://github.com/clauxel/MiroFish): a short front door, a clear reading order, practical guides, reference pages, and a public-safe boundary.
